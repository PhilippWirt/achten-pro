# Bilder-Ordner

## Was hier reinkommt:

- `isar-flaucher.jpg` — Hero-Bild der Website
  - Download von: https://upload.wikimedia.org/wikipedia/commons/2/25/Flaucher_Isar_Sommer_Muenchen-7.jpg
  - Oder eigenes besseres Isar-Foto

- `philipp-portrait.jpg` — Profilbild
  - Format: hochkant, möglichst natürlich + professionell
  - Auch für Sticker benötigt!

## Bilder herunterladen

Isar-Bild herunterladen:
```powershell
Invoke-WebRequest -Uri "https://upload.wikimedia.org/wikipedia/commons/2/25/Flaucher_Isar_Sommer_Muenchen-7.jpg" -OutFile "isar-flaucher.jpg"
```
